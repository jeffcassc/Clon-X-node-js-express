#  Twitter Clone - MERN Stack

Un clon de Twitter/X construido con MERN Stack para practicar desarrollo fullstack.

##  Características

-  **Autenticación** - Registro e inicio de sesión con JWT
-  **Tweets** - Crear, eliminar tweets (máx. 280 caracteres)
-  **Likes** - Dar y quitar likes a tweets
-  **Comentarios** - Comentar en tweets
-  **Perfiles** - Ver y editar perfil de usuario
-  **Seguir** - Seguir y dejar de seguir usuarios
-  **Búsqueda** - Buscar usuarios



##  Tecnologías

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT
- bcryptjs
